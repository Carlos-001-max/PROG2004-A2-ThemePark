# PROG2004 Assignment 2 - Theme Park Management System

## Project Overview
This Java application manages theme park operations including visitor tracking, ride management, and employee assignments.

## Features
- Object-oriented design with inheritance and polymorphism
- Queue management for ride waiting lines
- LinkedList for ride history tracking
- File I/O operations for data persistence
- Custom exception handling
- Comparator-based sorting

## Classes
- Person (abstract base class)
- Employee and Visitor (subclasses)
- Ride (implements RideInterface)
- VisitorComparator (for sorting)
- ThemeParkException (custom exceptions)

## How to Run
1. Compile all Java files: javac src/*.java
2. Run main class: java -cp src AssignmentTwo
